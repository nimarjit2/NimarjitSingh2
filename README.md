📊 Financial Regression Analysis using Python

Continuing my exploration of financial data, I performed a multiple regression analysis on the balance sheet of Ashok Leyland to understand how different financial components influence overall asset structure.

Using PyCharm for development , I built a regression model and visualized the regression coefficients of key balance sheet variables.

🔎 What this analysis includes:

Building a linear regression model on balance sheet variables

Interpreting coefficients, p-values, and statistical significance

Visualizing how financial variables contribute positively or negatively to the model

📈 Insights from the model:

Certain variables such as Total Liabilities and Total Assets show strong positive influence in the regression model.

Variables like Investments, CWIP, and Other Assets display negative coefficients, indicating inverse relationships within the model.

The statistical summary also highlights multicollinearity considerations, which is common when working with closely related financial variables.

💡 Key takeaway:
Applying regression analysis to financial statements helps uncover deeper relationships between financial indicators and supports data-driven financial interpretation.

This project further strengthened my experience in financial analytics, statistical modeling, and Python-based data visualization.
